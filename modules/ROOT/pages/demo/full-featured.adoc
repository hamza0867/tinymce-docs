The following plugins are excluded from this example:

* link:{baseurl}/plugins/premium/moxiemanager[MoxieManager (Premium Plugin)] -- link:{baseurl}/plugins/premium/tinydrive[*{cloudfilemanager}*] included instead.
* link:{baseurl}/plugins/opensource/autoresize[Autoresize] -- Resizes the editor to fit the content.
* link:{baseurl}/plugins/opensource/bbcode[BBCode] -- Changes the markup used for the content.
* link:{baseurl}/plugins/opensource/code[Code] -- link:{baseurl}/plugins/premium/advcode[*Advanced Code Editor*] included instead.
* link:{baseurl}/plugins/opensource/fullpage[Full Page] -- Used for modifying HTML `<head>` elements.
* link:{baseurl}/plugins/opensource/paste[Paste] -- link:{baseurl}/plugins/premium/powerpaste[*PowerPaste*] included instead.
* link:{baseurl}/plugins/opensource/spellchecker[Spellchecker] -- link:{baseurl}/plugins/premium/tinymcespellchecker[*Spell Checker Pro*] included instead.
* link:{baseurl}/plugins/opensource/tabfocus[Tab Focus] -- Changes the behavior of the TAB-key within the editor.
